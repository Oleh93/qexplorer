# qexplorer

### Prerequisites

```
git clone https://github.com/Oleh93/qexplorer
```

### Building

#### Linux/MaxOS
```
cmake -G "Unix Makefiles"
make
```
#### Windows
```
cmake .
make
```

### Running
```
./qexplorer
```
